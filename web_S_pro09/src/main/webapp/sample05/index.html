윈도우 크기 설정
var newWindow = window.open("about:blank(윈도우 링크)", "_blank", "width= 500", "height= 500");
newWindow.moveTo(0가로축,0세로축) -> 윈도우 창이 0,0위치로 옮겨짐(절대위치))
newWindow.resizeTo(500, 500); -> 창크기 조절
newWindow.resizeTo(100, 100) -> 창크기 누적(기존창에서 추가로 변경)
newWindow.focus(); newWindow.blur(); -> 포커스주고 블러하고~ 창을 띄우고 뒤로보내고 차이
newWindow.setTimeout(() => {return alert("hello")}, 3000); -> 일정시간 (3초)뒤에 함수를 실행해라
newWindow.setInterval(alert, 3000);
newWindow.clearInterval(0); -> 위의 setInterval을 clear

window.sesizeBy(100,100); -> (chrome)자신의 창은 안댐 새로운 창을 만들어서 조정만 가능
